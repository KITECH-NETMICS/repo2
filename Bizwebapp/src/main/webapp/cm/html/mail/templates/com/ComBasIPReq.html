신규 IP발급 신청 완료

요청하신 IP발급 신청이 다음과 같이 완료되었습니다.

◈ IP발급 신청 내용.
신청번호 @reqNo@
컴퓨터이름 @computerName@
발급IP주소 @ipAddress@
서브넷마스크 @subnetMask@
기본 게이트웨이 @primaryGateway@
기본 DNS @primaryDNS@
보조 DNS @secondaryDNS@
비고 @note@
  • 컴퓨터 이름은 사용자 이름으로 설정하는 컴퓨터실명제를 준수하시기 바랍니다(예시: 홍길동 또는 홍길동-PC). 컴퓨터실명제 미준수시 네트워크가 차단될 수 있습니다. ☞컴퓨터이름 변경방법
  • 발급된 IP는 해당기기에서만 사용가능합니다. 다른 기기에서 사용할 경우 차단됩니다. PC를 교체하는 경우에는 IP변경신청을 하시기 바랍니다.
