method String

[default: get, post, urlencoded-post]

  • get : 파라메타를 url에 붙이는 방식 (HTML과 동일).
  • post : 파라메타를 body 구간에 담는 방식 (HTML과 동일)
  • urlencoded-post : urlencoded-post.
mediatype String

[default: application/json, application/xml, text/xml, application/x-www-form-urlencoded] application/x-www-form-urlencoded 웹 form 방식(HTML방식). application/json : json 방식. application/xml : XML 방식. text/xml : xml방식 (두 개 차이는 http://stackoverflow._com/questions/4832357 참조)
mode String

[default: synchronous, synchronous] 서버와의 통신 방식. asynchronous:비동기식. synchronous:동기식
